# uniswap-v3-insights-dashboard


Dynamic on-chain analytics dashboard providing insights into Uniswap V3 pools, swaps, liquidity, and fees with rolling 30-day updates.

## What this covers
- Swap volume and activity (last 30 days)
- Liquidity distribution across pools
- Fee generation trends
- DEX performance insights over time

## Tools
- Dune Analytics
- SQL
- Uniswap V3 on-chain data

## Live Dashboard
https://dune.com/marviszn/uniswap-v3-insights-dashboard

## Update frequency
- Automatically updates on a rolling 30-day interval

## Use cases
- DeFi analytics
- DEX research
- Liquidity and LP behavior analysis
- Web3 data collaborations
